Примеры использования API

Получение списка постов

curl -X GET "https://api.socialplus.com/v1/posts" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json"

Этот запрос вернет список последних публичных постов в формате JSON.

Создание нового поста

curl -X POST "https://api.socialplus.com/v1/posts" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "content": "Привет, мир! Это пост, созданный через API.",
  "image_url": "https://example.com/image.jpg"
}'

Этот запрос создаст новый пост с указанным содержимым и изображением.

Получение информации о пользователе

curl -X GET "https://api.socialplus.com/v1/users/username" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json"

Этот запрос вернет публичную информацию о пользователе с указанным именем пользователя.
